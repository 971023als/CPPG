# GDPR에서의 용어 정리 (2022년 출제)

| **용어**       | **정의**                                                                 |
| --------------- | ------------------------------------------------------------------------ |
| **개인정보**    | 식별되었거나 식별 가능한 자연인과 관련된 모든 정보                       |
| **컨트롤러**    | 개인정보 처리의 목적과 수단을 스스로 결정하는 주체 (개인정보처리자 유사) |
| **프로세서**    | 컨트롤러를 대신하여 개인정보를 직접 처리하는 자연인, 법인, 정부부처, 관련기관 등 |
| **프로파일링**  | 개인의 특성을 평가하기 위하여 행해지는 모든 형태의 자동화된 개인정보의 처리 |
